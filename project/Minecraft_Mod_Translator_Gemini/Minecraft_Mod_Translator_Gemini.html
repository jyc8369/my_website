Minecraft Mod Translator Gemini (MMTG)

Overview

Minecraft 모드 JAR 파일 내부의 lang/*.json 언어 파일을 탐색하고, Google Gemini API로 번역한 뒤 다시 JAR로 패키징하는 도구입니다.

Runtime Python 3.12+와 타입 힌트 기반 모듈 분리 구조를 사용합니다.
GUI customtkinter로 현대적인 데스크톱 UI를 제공합니다.
AI Translation google-genai로 key는 유지하고 value만 번역합니다.
Packaging PyInstaller 및 release/build 스크립트로 배포 패키지를 생성합니다.

Tech Stack

Runtime Python 3.12+
GUI customtkinter
AI API google-genai
Build PyInstaller
Core Libraries logging, threading, json, zipfile, pathlib, shutil

Core Flow

프로그램은 입력부터 결과 저장까지 아래 순서로 동작합니다.

JAR 또는 폴더 선택 -> 임시 디렉터리 압축 해제 -> 언어 파일 탐색 -> 번역 대상 선정 -> Gemini 번역 -> 파일 교체 -> JAR 재압축 -> 결과 저장

JAR 처리 입력이 JAR이면 먼저 임시 폴더로 풀어서 내부 리소스를 분석합니다.
JSON 처리 key는 유지하고 value만 번역한 뒤 원본 구조를 보존합니다.
Result 번역 완료 후 다시 압축해 결과를 저장합니다.

Gemini Translation

GitHub Repository
Chunking 큰 JSON 파일은 MAX_CHUNK_CHARS와 MAX_CHUNK_ITEMS 기준으로 나눠 보냅니다.
Retries MAX_RETRIES와 RETRY_BASE_DELAY 기반의 재시도 로직을 사용합니다.
Prompt Rules JSON 키 변경 금지와 Minecraft 플레이스홀더 보존 규칙을 명시합니다.

Config & UI

Config Path Windows, macOS, Linux의 표준 설정 경로에 MMTG/config.json을 저장합니다.
Auto Creation 초기 실행 시 설정 파일이 없으면 자동 생성합니다.
Async UI 번역 작업은 threading으로 백그라운드 실행합니다.
Logging 커스텀 logging handler로 로그를 GUI 창에 실시간 출력합니다.
i18n 한국어와 영어 UI를 지원합니다.

Build & Deployment

Windows release/build/build_windows.bat
macOS release/build/build_macos.sh
Linux Packages Ubuntu/Debian은 .deb, Fedora는 .rpm, Arch는 .pkg.tar.zst를 생성합니다.
Cleanup 빌드 후 build/ 폴더를 삭제합니다.

요구 사항은 Python 3.12+와 pip install -r requirements.txt 입니다.
